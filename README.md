# 🧮 Play With Maths

A fun and interactive math learning app built with Flutter
where students can practice and improve math skills!

## 🌐 Live Demo
🔗 [play-with-maths.vercel.app](https://play-with-maths.vercel.app)

## 🔥 Features
- 🧮 Interactive math problems
- 📊 Progress tracking
- 👤 User authentication
- 🏆 Score system
- 📱 Cross-platform (Mobile + Web)

## 🛠️ Tech Stack
| Layer | Technology |
|-------|-----------|
| Frontend | Flutter (Dart) |
| Backend | Node.js, Express.js |
| Database | MongoDB |
| Deployment | Vercel |

## ⚙️ How to Run Locally

### Backend
```bash
cd backend
npm install
node index.js
```

### Frontend
```bash
cd frontend
flutter pub get
flutter run
```

## 🌐 API Endpoints
| Method | Endpoint | Description |
|--------|----------|-------------|
| POST | /api/signup | User registration |
| POST | /api/login | User login |
| GET | /api/scores | Get scores |
| POST | /api/submit | Submit answer |
